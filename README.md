## Server Setup

The backend code is located in the `backend` folder.

### Steps to Run the Backend

1. Navigate to the backend directory:  
   ```bash
   cd backend
   ```

2. **Install Dependencies:**
   ```bash
   npm install 
   ```

3. **Start the Server:**   
   ```bash
   npm start 
   ```

   The server listens on port 7070.
